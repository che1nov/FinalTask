# BUG-002: Проверка достаточности средств использует рублёвый баланс для всех валют

## Описание
При выборе счёта «Доллары» или «Евро» проверка возможности перевода всё равно считается от параметров рублёвого баланса из URL.

## Окружение
- URL: `/?balance=0&reserved=0`
- Браузер: Chrome

## Предусловия
Открыта главная страница сервиса.

## Шаги воспроизведения
1. Выбрать счёт «Доллары» (баланс 100$).
2. Ввести карту `1234 5678 9012 3456`.
3. Ввести сумму `50`.

## Ожидаемый результат
Кнопка «Перевести» доступна, так как на долларовом счёте хватает средств.

## Фактический результат
Показывается сообщение «Недостаточно средств на счете», перевод недоступен.

## Серьёзность
Major

## Приоритет
High

## Предполагаемая причина
В расчёте используется только `balance`/`reserved` из URL (рубли), без учёта выбранной валюты.

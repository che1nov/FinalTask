# BUG-001: Перевод блокируется при нулевом остатке после списания

## Описание
При переводе, где `сумма + комиссия` ровно равны доступному балансу, операция блокируется как недостаток средств.

## Окружение
- URL: `/?balance=2000&reserved=0`
- Браузер: Chrome

## Предусловия
Открыта главная страница сервиса.

## Шаги воспроизведения
1. Выбрать счёт «Рубли».
2. Ввести карту `1234 5678 9012 3456`.
3. Ввести сумму `1820`.

## Ожидаемый результат
Кнопка «Перевести» доступна (остаток 0 считается допустимым).

## Фактический результат
Показывается сообщение «Недостаточно средств на счете», перевод недоступен.

## Серьёзность
Major

## Приоритет
High

## Предполагаемая причина
Проверка выполнена по строгому условию `остаток > 0` вместо `остаток >= 0`.

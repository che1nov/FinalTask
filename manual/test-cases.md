# Ручные тест-кейсы

## Окружение
- ОС: macOS / Linux
- Браузер: Chrome
- URL сервиса: `http://localhost:4173`

## TC-001. Успешный перевод с рублёвого счёта при достаточном остатке
- Предусловия: открыть `/?balance=5000&reserved=200`.
- Шаги:
1. Выбрать счёт «Рубли».
2. Ввести номер карты `1234 5678 9012 3456`.
3. Установить сумму перевода `1000`.
4. Нажать «Перевести».
- Ожидаемый результат: кнопка перевода доступна, появляется подтверждение перевода.
- Фактический результат: соответствует ожидаемому.
- Статус: `Passed`.

## TC-002. Форматирование номера карты и фильтрация недопустимых символов
- Предусловия: открыть `/?balance=5000&reserved=0`.
- Шаги:
1. Выбрать счёт «Рубли».
2. Ввести строку `1234abcd5678!9012@3456`.
- Ожидаемый результат: в поле остаются только цифры и формат по 4 символа (`1234 5678 9012 3456`).
- Фактический результат: соответствует ожидаемому.
- Статус: `Passed`.

## TC-003. Блокировка перевода при недостатке средств
- Предусловия: открыть `/?balance=1000&reserved=900`.
- Шаги:
1. Выбрать счёт «Рубли».
2. Ввести номер карты `1234 5678 9012 3456`.
3. Ввести сумму `500`.
- Ожидаемый результат: отображается сообщение «Недостаточно средств на счете», кнопка «Перевести» недоступна.
- Фактический результат: соответствует ожидаемому.
- Статус: `Passed`.

## TC-004. Граничное условие: остаток после комиссии равен нулю
- Предусловия: открыть `/?balance=2000&reserved=0`.
- Шаги:
1. Выбрать счёт «Рубли».
2. Ввести номер карты `1234 5678 9012 3456`.
3. Ввести сумму `1820` (комиссия 180, итог 2000).
- Ожидаемый результат: перевод разрешён (остаток 0 допустим), кнопка «Перевести» доступна.
- Фактический результат: отображается «Недостаточно средств на счете», кнопка перевода отсутствует.
- Статус: `Failed`.
- Дефект: `BUG-001`.

## TC-005. Проверка средств должна выполняться в выбранной валюте
- Предусловия: открыть `/?balance=0&reserved=0`.
- Шаги:
1. Выбрать счёт «Доллары» (доступно 100$).
2. Ввести номер карты `1234 5678 9012 3456`.
3. Ввести сумму `50`.
- Ожидаемый результат: перевод разрешён, так как на долларовом счёте достаточно средств.
- Фактический результат: отображается «Недостаточно средств на счете», кнопка перевода отсутствует.
- Статус: `Failed`.
- Дефект: `BUG-002`.
